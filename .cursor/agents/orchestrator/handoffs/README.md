# Handoffs

One-off handoffs are not stored here. Use the **handoff template** in the parent directory for the standard format:

- **Template**: `../handoff-template.md`

New scope and execution-plan docs go in the project `docs/` folder:
- **Execution plans** (multi-step, phase-based): `docs/execution-plans/` (e.g. `docs/execution-plans/phase-X-feature-name.md`).
- **Scope docs** (feature/area context for handoffs): `docs/scope/` (e.g. `docs/scope/feature-name.md`).
If those folders don't exist yet, create them and add a short README in each (see Phase 3.2 in `.cursor/PLAN-cursor-next-steps.md`).
