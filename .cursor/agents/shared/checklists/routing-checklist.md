# Routing Checklist

- Confirm task phase: discover, plan, implement, verify, release, or document.
- Confirm primary artifact: product scope, code, schema, infra, tests, docs.
- Assign single primary owner agent.
- Decide `single`, `sequential`, or `parallel` delegation.
- Define acceptance criteria per delegated task.
- Attach contract template in every handoff.
- Assign merge owner if multiple child agents are used.
- Stop delegation if no new evidence appears.
