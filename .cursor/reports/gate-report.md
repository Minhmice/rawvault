# Gate Report

Generated by the **stop** hook (gate-runner). Treat this as the source of truth for gate status.

When the hook runs, it will overwrite this file with:
- **QA (tests):** npm test result (pass/fail, output summary)
- **Lint / typecheck:** npm run lint result
- **Security (quick scan):** Simple secret pattern scan result
- **Docs:** Manual (ensure how-to test and config/env notes exist)

**Ready to merge when:** All gates above are PASS/REVIEW and no pending handoffs in handoffs.md.
